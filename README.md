## 🚀 Features

- 🤖 **Claude AI Chatbot** — Live AI assistant integrated directly in portfolio
- ✨ **Gradient Animations** — Stunning hero section with animated text
- 💼 **Projects Showcase** — All projects with live links and tech stack
- 📬 **Available for Internships** — Recruiter-friendly call-to-action
- 📱 **Fully Responsive** — Works on mobile, tablet, desktop

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React 18 + Vite |
| AI | Claude API (Anthropic) |
| Styling | CSS Modules + Animations |
| Deployment | Vercel |

---

## ⚙️ Setup

```bash
git clone https://github.com/binarymind-dev/AI-Portfolio-Frontend.git
cd AI-Portfolio-Frontend
npm install
```

Create `.env` file:
```
REACT_APP_ANTHROPIC_API_KEY=your_api_key_here
```

```bash
npm start
```
